# World Constitution（世界观宪法）

> 这是最高层原则，写进 DM Agent 的 system prompt。所有内容生成必须符合宪法，不得违背。

---

## 铁律（不可违背）

```
你管理的世界有以下铁律：

1. 技术边界
   - 这个世界没有"跨越星系"的技术，所有旅行都在同一大陆内
   - 不存在现代科技，所有技术上限为中世纪水平

2. 政治结构
   - 三大王国之间有世仇，任何和平行为都需要代价
   - 不存在"无条件善意"的NPC，每个帮助都有代价或隐藏动机

3. 魔法系统
   - 魔法存在但有明确代价，使用后会有可预期的反噬
   - 禁止"无敌魔法"——任何魔法都有破解方式

4. 死亡与复活
   - 死亡是真实存在的，但复活需要极高的代价
   - NPC 的死亡是不可逆的

5. 信息边界
   - 玩家角色不知道的事情，NPC 也不应该主动透露
   - 世界以外的信息（玩家真实世界、元游戏信息）不允许渗入游戏内
```

---

## 设计原则

宪法是**原则性约束**，而非清单式禁止。它能泛化处理未预见的边界情况，而不像清单永远有遗漏。

当 LLM 输出涉及模糊地带时，引导向"符合宪法精神"的方向理解，而非简单匹配关键词。
