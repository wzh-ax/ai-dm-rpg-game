# 完整游戏流程

## 游戏阶段总览

```
【阶段一：准备阶段】
    ├── 房主创建房间（可选地图 + 难度）
    ├── 玩家创建/选择角色
    └── 玩家加入房间

【阶段二：新手引导】（仅第一次）
    └── 引导玩家了解基本操作

【阶段三：冒险开始】
    ├── 进入第一个 Beat
    ├── 主 DM 描述初始场景
    └── 玩家开始探索

【阶段四：循环探索】
    └── 重复 Beat 流程，直到副本结束

【阶段五：副本结束】
    ├── 主 DM 生成副本总结
    ├── 保存 checkpoint
    └── 玩家反馈收集（可选）
```

---

## Beat 流程详解

```
【Beat N 开始】
    主DM：
      → 描述当前场景/事件背景
      → 向各子Agent 发送本回合上下文 + 交互边界

【Beat N 进行中】
    子Agent 与玩家交互：
      → 接收玩家输入
      → 实时提醒剩余交互次数
      → 判断是否触发结束条件

【Beat N 结束】
    子Agent 汇报 → 主DM 裁决 → 状态更新
    ↓
【Beat N+1 开始】
```

---

## 退出条件

- 玩家主动结束副本
- 主线剧情完成（到达终点）
- 角色死亡（可选复活机制）
- 房主关闭房间
