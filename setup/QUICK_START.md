# 新手引导

## 目标

第一次游玩时，通过简短引导让玩家理解基本操作，之后直接开启即玩。

---

## 引导流程

### 第一次（强制引导）

```
欢迎来到 AI DM RPG！
你可以用自然语言与我对话，我会作为你的 DM 引导你进行冒险。

【基础操作】
- 输入你想做的事，DM 会描述结果
- 点击"结束回合"结束本轮行动
- 输入"帮助"查看所有可用命令

【开始冒险】
让我们从创建你的角色开始...
```

### 非第一次

直接进入 SESSION_FLOW，无需引导。

---

## 创角流程（可选简化）

MVP 阶段可极度简化：
- 玩家自选职业/背景关键词
- LLM 根据关键词生成角色描述
- 玩家确认或调整

---

## 帮助命令

| 命令 | 说明 |
|------|------|
| `帮助` | 显示所有可用命令 |
| `状态` | 查看当前角色状态 |
| `地图` | 查看已知区域 |
| `回顾` | 查看近期剧情摘要 |
