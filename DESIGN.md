## 项目概述

**项目名称**：AI DM RPG
**类型**：AI Dungeon Master 驱动的多人角色扮演游戏

**核心理念**：
- 用大语言模型（LLM）作为 DM（Dungeon Master），驱动游戏叙事和事件
- 玩家通过自然语言与 AI 交互，完成角色扮演冒险
- 支持多人联机，各玩家拥有独立视角和行动窗口

**与传统 D&D 的区别**：
- 不需要真人 DM，AI 承担叙事、控制 NPC、规则裁决等职责
- 内容为 AI 定制（非 D&D 原生），更利于 AI 理解和执行
- 支持多窗口交互模式（单人探索/多人讨论）

## 核心设计原则

1. **AI 作为叙事引擎**：LLM 驱动内容生成和剧情推进，非预设线性流程
2. **懒生成 + 分层审核**：大部分内容动态生成，核心内容人工把关
3. **Beat 机制统一回合**：无论是探索还是战斗，统一用"叙事节拍（Beat）"作为回合单位
4. **差异化生成**：同类型场景通过"先定性再生成"确保体验多样性
5. **宪法约束边界**：World Constitution 定义铁律，约束 LLM 生成不越界

## 最小闭环（MVP）

**目标**：跑通一个最小可玩的单人流程

**范围**：
- 单玩家 + 单子 Agent + 主 Agent
- 纯叙事，无战斗（或极简战斗）
- 文字输入
- 简单剧情跑通 Beat 机制 + 一次场景生成

**验证点**：
- Agent 协作流程是否通
- Hook 机制是否正常工作
- 状态存储/读取是否正确
