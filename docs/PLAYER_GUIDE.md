# AI DM RPG - 玩家指南

> 本游戏是由 AI 驱动的文字冒险 RPG，你扮演一名冒险者，在 AI DM（地下城主）的引导下探索世界、完成任务、体验剧情。

---

## 如何开始

```bash
cd C:\Users\15901\.openclaw\workspace\ai-dm-rpg
python interactive_master.py
```

---

## 游戏流程

### 第一步：创建角色

输入你的角色名字，选择种族和职业：

| 种族 | 说明 |
|------|------|
| 人类 | 均衡，无加成 |
| 精灵 | 敏捷高 |
| 矮人 | 体质强 |
| 兽人 | 力量大 |

| 职业 | 说明 |
|------|------|
| 战士 | 物理攻击强 |
| 游侠 | 敏捷型 |
| 法师 | 魔法攻击 |
| 盗贼 | 暴击率高 |

### 第二步：新手引导（Tutorial）

跟随 AI DM 的引导，了解基本操作。

### 第三步：开始冒险

你可以输入任何文字，AI DM 会理解并响应。

---

## 基本命令

| 输入 | 效果 |
|------|------|
| 任意文字 | AI DM 会作为描述或对话处理 |
| `状态` | 显示当前 HP、等级、位置、装备 |
| `帮助` | 显示可用命令 |
| `quit` / `exit` | 保存并退出游戏 |

---

## 战斗

当你遭遇敌人时，进入战斗模式：

| 输入 | 效果 |
|------|------|
| `攻击` | 普通攻击 |
| `防御` | 减少下次受到的伤害 |
| `技能` | 查看并使用职业技能 |
| `道具` | 查看背包并使用物品 |
| `逃跑` | 尝试脱离战斗 |

战斗结束后会根据结果获得经验值、金币和物品奖励。

---

## 难度模式

游戏支持三种难度：

| 难度 | 敌人 HP | 敌人伤害 | 掉落倍率 |
|------|---------|---------|---------|
| 简单 | ×0.7 | ×0.7 | ×1.5 |
| 普通 | ×1.0 | ×1.0 | ×1.0 |
| 困难 | ×1.5 | ×1.5 | ×0.8 |

---

## 存档与读档

游戏会自动保存进度。你可以在主菜单选择「继续游戏」加载存档。

---

## 提示

- **自由度很高**：你可以输入任何描述，AI 会尽力理解和响应
- **任务系统**：注意 NPC 的对话和场景描述，里面藏着任务线索
- **探索**：多和 NPC 说话、多去新场景，可能发现隐藏内容
- **升级**：积累经验提升等级，增强属性
